[justify]Раздел «Загрузки и шаблоны» предназначен для управления файлами, загруженными в директории модулей, в том числе: управление загруженными файлами, загрузка новых файлов, управление шаблонами вывода файлов на страницах сайта.

[b][color=green]Вкладка «Каталог файлов»[/color][/b]
В данной вкладке можно:
[li] Загрузить файлы в директорию выбранного модуля (директория/модуль выбирается в правом верхнем углу).
[li] Управлять списком файлов, загруженных в основную директорию модуля.
[b][i]Основная директория[/i][/b] – файлы, загруженные администратором, посетителями и пользователями в зависимости от выбранного модуля
[li] Управлять списком файлов, загруженных в дополнительную директорию модуля.
[b][i]Дополнительная директория[/i][/b] – как правило, это файлы предпросмотра (миниатюры), созданные системой в автоматическом режиме. Такая возможность есть при публикации картинок при помощи BB-редактора. Размер автоматического сокращения картинки (Ширины, Высоты) можно указать в конфигурациях, для каждого модуля персонально.

[color=green][b]Вкладка «Шаблоны»[/b][/color]
Для каждого типа файлов (типы файлов можно добавить во вкладке «Конфигурации» >> подраздел «Основные»), загруженного на сайт (как правило, при использовании BB-редактора), можно установить свой шаблон. Шаблон активен для вставленных в BB-редакторе объектов (Левая кнопка «Скрепка» в нижней панели редактора).

В зависимости от разрешения файла, при выводе будет использован автоматически тот или иной шаблон, установленный администратором. Возможна установка размера прилагаемого объекта, как ширины, так и высоты, в случае отсутствия размера, система использует установленные по умолчанию.
Администратор имеет возможность установки персонального шаблона для каждого формата, что в свою очередь даёт неограниченную возможность применения различных видов отображения графики, показа видео, проигрывания музыки и т.д.

[li] [b][i]align=center[/i][/b] – позиция объекта (left, right, center);
[li] [b][i]title=title[/i][/b] – альтернативный текст, заголовок;
[li] [b][i]width=500[/i][/b] – ширина;
[li] [b][i]height=500[/i][/b] – высота;
[li] [b][i]rel=rel[/i][/b] – если использовать, то покажет группу картинок с возможностью их перелистывания. Значение rel после знака равно может быть любое, для группы картинок желательно уникальное, допустим rel=my_img.

[color=green][b]Вкладка «Конфигурации»[/b][/color]
Содержит два подраздела:

[b][i][color=orangered]Основные[/color][/i][/b]
Задаются основные конфигурации для работы с загрузкой и отображением файлов:
[li] [b][i]Директория по умолчанию[/i][/b] – директория, которая будет открываться по умолчанию во вкладке «Главная» раздела «Загрузки и шаблоны».
[li] [b][i]Форматы приложений[/i][/b] – форматы, для которых можно будет добавить шаблон в отделе шаблоны.
[li] [b][i]Ширина приложения по умолчанию[/i][/b] – задается на случай, если в конфигурациях к модулю ничего не указано.
[li] [b][i]Высота приложения по умолчанию[/i][/b] – задается на случай, если в конфигурациях к модулю ничего не указано.

[b][i][color=orangered]Модули[/color][/i][/b]
Задаются конфигурации загрузок персонально для каждого модуля.
[hr][color=blue][b]Обратите внимание![/color][/b]
Модуль «Все» отвечает за модули, не имеющие панель управления администратором. К примеру, модуль пользователя или сторонние модули, работающие с BB-редактором.[/justify]
