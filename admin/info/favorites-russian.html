[justify]Раздел «Мои фавориты» предназначен для просмотра и управления списком материалов, добавленных пользователями сайта в фавориты. Также в разделе можно задать настройки для «Мои фавориты».

[color=green][b]Вкладка «Главная»[/b][/color]
Отображает список материалов-фаворитов, добавленных пользователями сайта. В списке отображается следующая информация:
[li] [b][i] Модуль[/i][/b] – к какому модулю принадлежит материал-фаворит.
[li] [b][i]Заголовок[/i][/b] – соответственно, заголовок материала-фаворита. Выводится в виде гиперссылки, переводящей к просмотру материала.
[li] [b][i]Разместил[/i][/b] – выводится ник пользователя, который добавил материал в свой список фаворитов.

С каждым материалом администратор может выполнить следующие действия (переход к действиям в столбце «Функции»):
[li] [b][i]Подробный просмотр[/i][/b] – открывает материал-фаворит на сайте.
[li] [b][i]Удаление объекта[/i][/b] – удаляет закладку на материал из раздела «Мои фавориты» (у пользователя, добавившего фаворит, тоже удаляется).

[color=orangered][b][i]Cколько раз материал был добавлен в фавориты[/i][/b][/color]
В таблице с материалами-фаворитами можно производить сортировки по содержимому столбцов, нажав на заголовок столбца. Отсортировав данные по столбцу «Заголовок» можно увидеть, сколько раз определённый материал был добавлен в фавориты.

[color=green][b]Вкладка «Конфигурации»[/b][/color]
В данной вкладке задаются следующие настройки для «Мои фавориты»:
[li] [b][i]Количество публикаций на страницу пользователя[/i][/b] – количество материалов, добавленных в фавориты, которое будет выводиться на одной странице в личном кабинете пользователя.
[li] [b][i]Количество публикаций на страницу администратора[/i][/b] – количество материалов, добавленных в фавориты, которое будет выводиться на одной странице в разделе «Мои фавориты» панели управления.
[li] [b][i]Максимальное количество номеров страниц пользователей (администраторов)[/i][/b] – список страниц будет включать указанное число страниц, а после будет выводиться отделённый многоточием номер последней страницы.
[li] [b][i]Максимально допустимое количество[/i][/b] – максимально количество материалов, которые один пользователь может добавить в фавориты.
[li] [b][i]Активировать фавориты?[/i][/b] – с помощью данной настройки можно отключить модуль «Мои фавориты».[/justify]
