[justify]Раздел «Комментарии» предназначен для управления комментариями, размещаемыми пользователями на сайте (модерирование).

[b][color=green]Управление комментариями[/color][/b]
Управление комментариями осуществляется во вкладках:
[li] [b][i]Главная[/i][/b] – управление всеми размещёнными на сайте комментариями.
[li] [b][i]Ждут проверки[/i][/b] – управление комментариями, размещёнными «гостями» сайта (неавторизованные посетители). Комментарии «гостей» отправляются на проверку, если в конфигурации комментариев указана соответствующая опция.

[b][i][color=orangered]Управление размещёнными комментариями[/color][/i][/b]
Возможны следующие действия с каждым размещённым комментарием:
[li] [b][i]Подробный просмотр[/i][/b] – происходит открытие комментария на сайте (страница материала, к которому добавлен комментарий).
[li] [b][i]Полное редактирование[/i][/b] – открывается форма изменения текста комментария.
[li] [b][i]Удаление объекта[/i][/b] – комментарий удаляется.

Для вызова какого-либо действия необходимо навести курсор мыши на кнопку (Редактор) (выводится в правом нижнем углу от комментария) и выбрать соответствующий пункт в отобразившемся возле кнопки меню.

Для [b][i]групповой обработки [/i][/b] комментарии можно отметить как по отдельности, поставив отметку в чек-боксе справа от комментария, так и все комментарии, отображаемые на странице, поставив отметку «Отметить все» справа от верхнего комментария.
Обратите внимание, что при переходе на следующую страницу списка комментариев отметки, поставленные на предыдущей странице, не сохраняются.
Действие с отмеченными комментариями выбирается снизу списка комментариев. Возможны следующие действия:
[li] [b][i]Активировать[/i][/b] – действие актуально для комментариев, ждущих проверки. При выборе данного действия комментарий выводится на сайте для всеобщего просмотра.
[li] [b][i]Удалить[/i][/b] – удаление отмеченных комментариев.

[b][i][color=orangered]Управление комментариями, ждущими проверки[/color][/i][/b]
Управление комментариями, ждущими проверки, аналогично управлению размещёнными комментариями.
Если комментарий подходит для размещения на сайте, то его необходимо отметить (чек-бокс справа) и выбрать действие с отмеченными (снизу списка справа) «Активировать».

[b][color=green]Конфигурации комментариев[/color][/b]
Управление настройками (конфигурациями) раздела «Комментарии» происходит во вкладке «Конфигурации», где можно задать следующие настройки:
[li] [b][i]Количество публикаций на страницу пользователя[/i][/b] – заданное количество комментариев будет отображаться на одной странице открытой части сайта. Рекомендуем не делать количество комментариев на одной странице более 30, чтобы не увеличивалось время загрузки страниц.
[li] [b][i]Количество публикаций на страницу администратора[/i][/b] – заданное количество комментариев будет отображаться на одной странице раздела «Комментарии» в административном разделе.
[li] [b][i]Максимальное количество номеров страниц пользователей[/i][/b] – если количество страниц с комментариями превышает заданное значение, то после номера страницы, соответствующего заданному числу, будет выводиться многоточие.
[li] [b][i]Максимальное количество номеров страниц администратора[/i][/b] – аналогично предыдущей настройке, но задается для раздела управления (административный раздел).

[b][color=green]Управление комментариями непосредственно на сайте[/color][/b]
Авторизовавшись в системе под учетной записью администратора, можно управлять комментариями непосредственно на сайте при просмотре прокомментированных материалов (не в административной части). Для администратора выводится кнопка (Редактор) (в левом верхнем углу от комментария), при наведении курсором мыши на которую отображается меню управления комментарием.
Меню содержит:
[li] [b][i]Быстрое редактирование[/i][/b] – без перезагрузки страницы комментарий будет отображён в режиме редактирования текста.
[li] [b][i]Закрыть на проверку[/i][/b] – комментарий будет переведён в статус «Ждёт проверки». Для удобства администратор видит на сайте комментарии со статусом «Ждёт проверки», но такие комментарии отображаются «бледнее» по сравнению с принятыми комментариями.
[li] [b][i]Активировать[/i][/b] – действие актуально для комментариев, ждущих проверки. При выборе данного действия комментарий выводится на сайте для всеобщего просмотра.[/justify]
