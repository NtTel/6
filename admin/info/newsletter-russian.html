[justify]Раздел «Рассылка» позволяет создавать информационные сообщения и рассылать их по группам пользователей (к примеру, новости проекта, информация о скидках, акциях и т.п.). письма отправляются исходя из посещаемости сайта: при запросе страницы сайта происходит отправка писем по активным рассылкам.

[color=green][b]Добавление рассылки[/b][/color]
Для добавления рассылки перейдите во вкладку «Добавить» и заполните информацию о рассылке:
[li] [b][i]Заголовок[/i][/b] – заголовок будет использован в качестве темы письма.
[li] [b][i]Текст[/i][/b] – текст письма, который будет разослан получателям рассылки.
[li] [b][i]Рассылку получат[/i][/b] – выбирается группа пользователей, которой будет отправлена рассылка. Справа от названия группы выводится количество пользователей в ней. Пояснение по некоторым группам:
[b] * [/b]Все пользователи – рассылка будет отправлена всем пользователям ресурса;
[b] * [/b]Только подписавшиеся пользователи – рассылка будет отправлена только тем пользователям ресурса, которые в своём личном кабинете указали значение «ДА» в настройке «Получать новости по почте?».
[b] * [/b]Клиенты модуля «Магазин» – рассылка только по пользователям, которые делали покупку через модуль «Магазин».
[li] [b][i]Количество писем в одном отправлении[/i][/b] – выбирается количество писем, которое будет отправляться за один раз. Рекомендуется устанавливать значение в 1-3 письма, чтобы почтовые системы не принимали рассылку за СПАМ и не блокировали адрес.
[li] [b][i]Начать отправку рассылки[/i][/b] – если выбрано «Да», то после выбора в списке рядом значения «Отправить» и нажатия кнопки «ОК» запустится рассылка. Если выбрано «Нет», то после выбора в списке рядом значения «Отправить» и нажатия кнопки «ОК» рассылка не запустится, а только сохранится.
[li] [b][i]Предварительный просмотр и отправка[/i][/b] – если в поле перед кнопкой «ОК» выбрано значение «Предварительный просмотр», то после нажатия кнопки не произойдёт рассылка, а в верхней секции формы добавления («Предварительный просмотр») отобразится текст рассылки так, как его увидят получатели. Если будет выбрано значение «Отправить» , то после нажатия кнопки запустится или только сохранится рассылка (в зависимости от значения параметра «Начать отправку рассылки»).

[color=green][b]Список рассылок[/b][/color]
Список всех рассылок, которые добавлялись, отображается во вкладке «Главная». Там же по каждой рассылке отображается её статус: Деактивирован или Активирован.
Статус «Деактивирован» присваивается уже разосланным рассылкам и рассылкам, сохранённым без отправки.
Статус «Активирован» присваивается рассылкам, которые в момент просмотра раздела ещё отправляются.

[li] Информацию о том, скольким пользователям отправлена рассылка и сколько времени длилась отправка можно просмотреть соответственно в столбцах: «Отправлено», «Время».
[li] Для просмотра даты и времени создания рассылки наведите курсор мыши на иконку (i) слева от заголовка нужной рассылки.
[li] Для сортировки списка рассылок по какому-то столбцу, нажмите на его название (при повторном нажатии произойдёт сортировка в обратном порядке).
[li] Для изменения рассылки нажмите в колонке «Функции» значение (Полное редактирование). Редактирование может пригодиться для повторной отправки рассылки или для запуска рассылки, если при добавлении она не была разослана.
[li] Для удаления рассылки нажмите в колонке «Функции» значение (Удаление объекта).

[hr][color=blue][b]Обратите внимание[/b][/color]
[li] Если рассылка была отредактирована и повторно отправлена, то она не добавляется в список рассылок новой строкой, а информация о статистике получателей и времени рассылки считается заново.
[li] При редактировании рассылки для повторной отправки убедитесь, что группа пользователей выбрана корректно.
[li] E-Mail, отображаемый в поле «От» задаётся в разделе «Конфигурации».[/justify]
