[justify]При помощи данного раздела можно самостоятельно и в простой форме расширить возможности информационной наполненности для следующих модулей: Профиль (профиль пользователя), Содержание, Форум, Помощь, Новости.
[hr][color=blue][b]Обратите внимание![/color][/b]
Дополнительные поля для модуля форум работают только в системном форуме (в форуме SLAED). В интегрированных форумах (к примеру, phpbb) дополнительные поля не отображаются.[hr]
Для добавления дополнительного поля в нужный модуль перейдите во вкладку, соответствующую названию модуля. Можно добавить поля следующих типов:
[li] Одна строка;
[li] Текстовое поле;
[li] Список с выбором;
[li] Дата: год-месяц-число часы:минуты (к примеру, 01-11-1900 12:34);
[li] Дата: год-месяц-число (к примеру, 01-11-1900).

Форма добавления дополнительного поля содержит:
[li] [b][i]Название[/i][/b] – укажите в данном поле название дополнительного поля.
[li] [b][i]Содержание[/i][/b] – можно указать содержание поля, которое будет выводиться в нем по умолчанию. Для поля «список с выбором» элементы списка записываются через запятую без пробела.
[li] [b][i]Тип поля[/i][/b] - выберите необходимый тип поля.
[li] [b][i]Использование[/i][/b] – выберите является ли поле обязательным или нет.
Для добавления нового дополнительного поля щелкните на знак плюс возле последнего добавленного поля.
Для удаления незаполненного нового поля – щелкните на знак плюс возле поля, стоящего перед незаполненным новым полем.
Для удаления заполненного и ранее сохраненного поля необходимо в поля «Название» и «Содержание» записать значение 0.
Учтите, что все изменения вступят в силу после того как будет нажата кнопка «Сохранить изменения».[/justify]
