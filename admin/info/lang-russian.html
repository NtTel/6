[justify]Раздел предназначен для управления служебными текстами сайта, выводящимися во всех модулях сайта во всех языковых версиях.
[b][color=green]Управление текстами[/color][/b]
Управления текстами языковых модулей происходит во вкладке «Главная», содержащей список модулей, в которых можно задать служебные тексты для языковых версий.
Пояснения по таблице, отображаемой во вкладке «Главная»:
[li] [b][i]Модуль[/i][/b] – отображается наименование модуля системы.
[li] [b][i]Кто видит[/i][/b] – отображается видят ли информацию из данного модуля все посетители сайта или только пользователи (авторизованные посетители).
[li] [b][i]Статус[/i][/b] – активирован ли в данный момент модуль (Активирован), то есть выводится на сайте, или нет (Деактивирован).

С каждым материалом администратор может выполнить следующие действия (переход к действиям в столбце «Функции»):
[li] [b][i]Администратор[/i][/b] – переход к редактированию служебных слов, используемых в разделе управления, или отсутствие значения, информирующая о том, что администраторская часть модуля не содержит служебных слов.
[li] [b][i]Модуль[/i][/b] – переход к редактированию служебных слов, используемых в открытой части системы, или отсутствие значения, информирующая о том, что пользовательская часть модуля не содержит служебных слов.

Переход к форме редактирования служебных слов для языковых версий осуществляется нажатием на значение в соответствующем столбце (Администратор или Модуль).

[hr][b][color=blue]Обратите внимание![/color][/b]
Для редактирования основных языковых констант панели администратора, а так же пользовательской частей требуется установленные в настройках PHP параметры Max input vars: 4000 (Максимально допустимое количество входящих переменных) и выше. Данный параметр доступен в блоке администратора «Системная информация».[hr]
[color=orangered][b][i]Правила задания языковых версий для контентных страниц (новости, тексты и другие)[/i][/b][/color]
Чтобы создать новости для какой-то языковой версии сайта (к примеру, для английской) необходимо создать категорию в разделе «Категории» и указать, в какой языковой версии она должна отображаться (категорию создавать соответственно на английском языке). Далее при создании англоязычной новости закрепите её за предварительно созданной языковой категорией. Пользователь при переходе к английской версии будет видеть соответственно новости, закрепленные за английской категорией.
Для создания языковых контентных страниц просто создавайте их содержимое на соответствующем языке и ссылку на них добавьте в нужной языковой версии.[/justify]
