[justify]Модуль «Каталог файлов» предназначен для публикации в соответствующем разделе сайта файлов.

[b][color=green]Вкладка «Главная»[/color][/b]
Во вкладке отображается перечень уже созданных страниц с возможностями:
[li] [b][i]Подробный просмотр[/i][/b] – открывается страница просмотра файла так, как она выглядит на сайте;
[li] [b][i]Полное редактирование[/i][/b] – открывается форма редактирования данных о файле;
[li] [b][i]Удаление объекта[/i][/b] – происходит удаление файла.

[b][color=green]Вкладка «Добавить»[/color][/b]
Чтобы добавить новый файл перейдите во вкладку «Добавить» и заполните форму добавления. Можно выбрать любой из следующих вариантов загрузки файла:
[li] [b][i]Загрузить файл с компьютера[/i][/b] – выбирается файл с компьютера (или с локальной сети);
[li] [b][i]Загрузить файл по ссылке[/i][/b] – укажите ссылку, по которой Системе необходимо «забрать» файл;
[li] [b][i]Ссылка на файл[/i][/b] – при таком варианте файл не хранится на сервере, а скачивается по указанной ссылке.
[hr][color=blue][b]Обратите внимание![/color][/b]
При таком варианте размещения файла необходимо указать размер файла в байтах.[hr]
При добавлении файла можно выбрать опцию «Предварительный просмотр», которая означает, что после нажатия кнопки «Ок» будет отражен вывод заполненных полей. Для окончательного добавления выберите «Отправить» и нажмите «Ок».

[b][color=green]Вкладка «Новые»[/color][/b]
Модерация файлов происходит во вкладке «Новые».
Новые файлы – это файлы, размещенные пользователями или гостями сайта (в случае, если это разрешено). Данные файлы не будут размещены в каталоге файлов, до тех пор, пока администратор не зайдет в форму редактирования данных файлов и не сохранит изменения.

[b][color=green]Вкладка «Недоступные файлы»[/color][/b]
Если при просмотре файла кто-то укажет, что файл недоступен, то такой файл будет выводиться в списке недоступных файлов во вкладке «Недоступные файлы». Администратор может:
[li] Просмотреть файл и проверить доступность, нажав на значение (Подробный просмотр).
[li] Игнорировать предупреждение, нажав на иконку значение (Игнорировать). После выполнения данного действия файл больше не будет выводиться в реестре недоступных файлов (до следующего предупреждения).

[color=green][b]Вкладка «Конфигурации»[/b][/color]
Управление настройками модуля происходит во вкладке «Конфигурации». Рассмотрим настройки, по которым могут возникнуть вопросы:
[li] [b][i]Максимально допустимый размер файла для загрузки, в байтах[/i][/b] – данный параметр устанавливает ограничение на размер загружаемого файла. Обратите внимание, что в поле можно ввести только число (варианты, 56 Кб или 5 Мб не подходят).
[li] [b][i]Допустимые форматы файлов для загрузки[/i][/b] – форматы файлов, которые можно загрузить при добавлении файла (перечисляются через запятую без пробела). Обратите внимание, что при попытке загрузить файл, формат которого не указан в данном поле, Система выдаст соответствующее предупреждение.
[li] [b][i]Количество колонок при выводе категорий[/i][/b] – файлы можно разбить по категориям; категории в интерфейсе сайта выводятся таблично. Данный параметр отвечает за количество колонок с категориями.
[li] [b][i]Разрешить добавлять файлы?[/i][/b] – смогут ли авторизованные пользователи сайта добавлять файлы.
[li] [b][i]Разрешить гостям добавлять файлы?[/i][/b] – смогут ли гости сайта добавлять файлы.
[li] [b][i]Возможность сообщать о недоступных файлах?[/i][/b] – будет ли при просмотре файла возможность сообщить о недоступности файла. Файлы, при просмотре которых сообщали о недоступности, попадают в список недоступных файлов.[/justify]
