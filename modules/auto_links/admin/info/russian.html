[justify]Модуль позволяет обмениваться ссылками с другими сайтами.

[b][color=green]Назначение обмена ссылками[/color][/b]
Как правило, обмен ссылками применяется в следующих целях:
[li] повышение Индекса Цитирования своего собственного сайта (полезно для поисковой оптимизации);
[li] привлечение дополнительных посетителей, приходящих с «дружественных сайтов».

[b][color=green]Добавление ссылок[/color][/b]
Ссылку в модуль «Обмен ссылками» можно добавить двумя способами:
1. Добавляет заинтересованное лицо (к примеру, администратор «дружественного» сайта) через раздел «Обмен ссылками» публичной части сайта.
2. Добавляет администратор непосредственно в разделе управления. Для этого необходимо перейти во вкладку «Добавить» и заполнить информацию по новой ссылке.
Все добавленные ссылки отображаются во вкладке «Главная».
При добавлении ссылки администратором можно задать начальные значения показателей:
[li] [b][i]Входы[/i][/b] – начальное количество переходов с добавляемого сайта;
[li] [b][i]Выходы[/i][/b] – начальное количество переходов с сайта, разработанного на SLAED CMS, на «дружественный» сайт.
Также данные о количестве входов/выходов можно задать на форме редактирования данных о ссылке. Переход к форме редактирования осуществляется во вкладке «Главная» путём нажатия значение (Полное редактирование) в столбце «Функции».

[color=green][b]Просмотр переходов, осуществленных по ссылке[/b][/color]
Чтобы просмотреть, когда и кто (IP) совершал переход по ссылке на сайт, необходимо нажать на значение (Подробный просмотр) напротив нужной ссылки в столбце «Функции» вкладки «Главная». При этом отобразится соответствующая информация из раздела «Переходы».

[color=green][b]Вкладка «Удалить сайты без входов»[/b][/color]
При переходе в данную вкладку происходит автоматическая чистка списка ссылок. При чистке удаляются все ссылки, с которых не было ни одного перехода на сайт.

[color=green][b]Вкладка «Обнулить значения»[/b][/color]
При переходе в данную вкладку автоматически обнуляется информация о количестве входов/выходов по ссылкам.

[color=green][b]Вкладка «Конфигурации»[/b][/color]
Во вкладке задаются настройки модуля.
Пояснения по некоторым настройками:
[li] [b][i]Установка баннера[/i][/b] – выбирается изображение (баннер), которое будет использоваться в html-коде баннера, предназначенного для вставки на «дружественный» сайт. Баннеры хранятся в директории: /images/banners/ (актуальной темы сайта).
[li] [b][i]Количество символов в ссылке блока[/i][/b] – в стандартном блоке обмена ссылками block_auto_links будет выводиться указанное количество символов по каждой ссылке (после выводится многоточие). Включение/отключение блока происходит в разделе «Блоки и баннеры».
[li] [b][i]Количество ссылок в блоке[/i][/b] – в стандартном блоке обмена ссылками block_auto_links будет выводиться указанное количество ссылок. Включение/отключение блока происходит в разделе «Блоки и баннеры».
[li] [b][i]Уведомлять администраторов о новых поступлениях?[/i][/b] – при добавлении новой ссылки администратору на e-mail будет приходить соответствующее уведомление.[/justify]
