[justify]Опросы способны «освежить» любой сайт (если, конечно, они интересно составлены). В модуле «Опросы» можно управлять опросами, которые будут выводиться на сайте. SLAED CMS позволяет связывать опросы с другими модулями системы (Новости, Магазин) – подробное описание см. ниже.

[color=green][b]Вкладка «Главная»[/b][/color]
Управление опросами происходит во вкладке «Главная».
Администратор может:
[li] [b][i]Подробный просмотр[/i][/b] – открывается страница с текущими итогами опроса. Кнопка доступна только в том случае, если опрос ещё не завершён;
[li] [b][i]Полное редактирование[/i][/b] – открывается форма редактирования опроса (форма редактирования аналогична по своей структуре форме добавления);
[li] [b][i]Удаление объекта[/i][/b] – происходит удаление опроса.

[color=green][b]Вкладка «Добавить»[/b][/color]
Для добавления опроса перейдите во вкладку «Добавить» и заполните форму, которая содержит:
[li] [b][i]Модуль[/i][/b] – можно указать, с каким модулем связан опрос (Новости или Магазин), что позволяет при добавлении записи в связанный с опросом модуль выбрать опрос, который будет отображаться при просмотре записи (к примеру, при просмотре определённых товаров магазина будет выдаваться определённый опрос);
[li] [b][i]Заголовок/Вопрос[/i][/b] – текст вопроса;
[li] [b][i]Варианты ответов[/i][/b] – можно указать неограниченное количество вариантов. Для добавления каждого нового варианта ответа необходимо нажать на иконку плюс слева от последнего добавленного варианта. Опрос подразумевает выбор как одного, так и нескольких вариантов ответа (настраивается). Для задания начального количества голосов какому-либо варианту используйте поле «Гол.», которое выводится справа от варианта ответа;
[li] [b][i]Дата публикации[/i][/b] – указывается дата, начиная с которой опрос станет доступен посетителям сайта;
[li] [b][i]Дата окончания[/i][/b] – указывается дата, начиная с которой участие в опросе будет невозможно;
[li] [b][i]После истечения[/i][/b] – выбирается вариант поведения опроса по истечении даты окончания: либо исчезать (деактивировать опрос), либо отображаться только для просмотра (отключить голосование);
[li] [b][i]Тип[/i][/b] – можно создавать опросы открытого (для всех) и закрытого (только для пользователей) типов;
[li] [b][i]Язык[/i][/b] – выбор языковой версии, в которой должен выводиться опрос;
[li] [b][i]Комментарии[/i][/b] – можно ли комментировать опрос или нет;
[li] [b][i]Разрешить возможность выбора нескольких вариантов?[/i][/b] – если выбрано «Нет», то это означает, что при ответе на опрос можно выбрать только один вариант, а если выбрано «Да» – можно выбрать несколько вариантов (чек-боксы).

[color=green][b]Вкладка «Конфигурации»[/b][/color]
Управление настройками модуля происходит во вкладке «Конфигурации». Рассмотрим настройки, по которым могут возникнуть вопросы:
[li] [b][i]Интервал участия в рейтинге, в сутках[/i][/b] – данный параметр обозначает: сколько дней пользователи смогут ставить опросу свою рейтинговую оценку, после первой.
[li] [b][i]Количество вариантов ответа[/i][/b] – максимально возможное количество полей для вариантов ответа при публикации опроса.
[li] [b][i]Отображать в блоке опросов[/i][/b] – данные настройки распространяются только на блок опросов. Для активации данного блока, войдите в отдел «Блоки и баннеры» панели администратора, активируйте или добавьте блок опросов, стандартное имя файла: block-voting.php[/justify]
