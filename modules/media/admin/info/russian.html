[justify]Модуль «Медиа каталог» предназначен для публикации в соответствующем разделе сайта описаний и ссылок на медиа файлы (к примеру, фильмы или музыкальные альбомы/композиции).

[b][color=green]Вкладка «Главная»[/color][/b]
Во вкладке отображается перечень уже созданных записей с возможностями:
[li] [b][i]Подробный просмотр[/i][/b] – открывается страница просмотра файла так, как она выглядит на сайте;
[li] [b][i]Полное редактирование[/i][/b] – открывается форма редактирования данных медиа файла;
[li] [b][i]Удаление объекта[/i][/b] – происходит удаление файла.

[b][color=green]Вкладка «Добавить»[/color][/b]
Чтобы добавить новый медиа файл перейдите во вкладку «Добавить» и заполните форму добавления.
[hr][color=blue][b]Обратите внимание![/color][/b]
[li] для добавления дополнительной ссылки на файл щелкните на знак плюс возле последней добавленной ссылки на файл. Для удаления незаполненной новой ссылки – щелкните на знак плюс возле поля с ссылкой, стоящего перед полем с незаполненной новой ссылкой.
[li] при добавлении файла можно выбрать опцию «Предварительный просмотр», которая означает, что после нажатия кнопки «Ок» будет отражен вывод заполненных полей. Для окончательного добавления выберите «Отправить» и нажмите «Ок».[hr][color=green][b]Вкладка «Новые»[/b][/color]
Модерация медиа файлов происходит во вкладке «Новые».
Новые медиа файлы – это файлы, размещенные пользователями или гостями сайта (в случае, если это разрешено). Данные медиа файлы не будут размещены в каталоге, до тех пор, пока администратор не зайдет в форму редактирования данных медиа файлов и не сохранит изменения.

[color=green][b]Вкладка «Недоступные медиа файла»[/b][/color]
Если при просмотре медиа файла кто-то укажет, что файл недоступен, то такой файл будет выводиться в списке недоступных медиа файлов во вкладке «Недоступные медиа файлы». Администратор может:
[li] Просмотреть медиа файл и проверить доступность, нажав на значение (Подробный просмотр);
[li] Игнорировать предупреждение, нажав на значение (Игнорировать). После выполнения данного действия медиа файл больше не будет выводиться в реестре недоступных медиа файлов (до следующего предупреждения).

[color=green][b]Вкладка «Конфигурации»[/b][/color]
Управление настройками модуля происходит во вкладке «Конфигурации». Рассмотрим настройки, по которым могут возникнуть вопросы:
[li] [b][i]Языки для выбора[/i][/b] – на форме добавления файла есть поле выбора языка файла. Возможные варианты языков задаются в данном поле с разделением через запятую.
[li] [b][i]Формат для выбора[/i][/b] – на форме добавления файла есть поле выбора формата файла. Возможные варианты форматов задаются в данном поле с разделением через запятую.
[li] [b][i]Качество для выбора[/i][/b] – на форме добавления файла есть поле выбора уровня качества файла. Возможные варианты уровня качества задаются в данном поле с разделением через запятую.
[li] [b][i]Максимальное количество ссылок[/i][/b] – сколько ссылок для скачивания можно разместить при описании одного медиа файла.
[li] [b][i]Разрешить добавлять медиа файлы?[/i][/b] – смогут ли авторизованные пользователи сайта добавлять медиа файлы.
[li] [b][i]Разрешить гостям добавлять медиа файлы?[/i][/b] – смогут ли гости сайта добавлять медиа файлы.
[li] [b][i]Возможность сообщать о недоступных медиа файлах?[/i][/b] – будет ли при просмотре медиа файла возможность сообщить о недоступности файла. Медиа файлы, при просмотре которых сообщали о недоступности, попадают в список недоступных медиа файлов.
[li] [b][i]Разрешить гостям видеть ссылки на загрузку медиа файлов?[/i][/b] – если установить «Нет», то ссылки на загрузку будут видеть только авторизованные пользователи.[/justify]
